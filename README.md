# **为什么会有这个网站**
---

>  本站参考 <a href="https://notebook.js.org/">r2coding</a>，在此基础上进行了修改，用于搭建个人的知识库，
仅供学习使用。
>  
>  旨在为编程自学者提供一系列：
>  - **清晰的编程自学路线**
>  - **各编程岗位方向的知识点梳理**
>  - **靠谱的资源**
>  - **高效的工具**
>  - **务实的技术文章**
>
> 方便自己也方便他人。网站内容会保持**持续更新**，欢迎收藏品鉴！
