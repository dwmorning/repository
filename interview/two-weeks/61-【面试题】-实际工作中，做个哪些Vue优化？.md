> v-if 和v-show

```
v-if彻底销毁组件
v-show使用css隐藏组件
大部分情况下使用v-if更好，不要过度优化
```

> v-for使用key（最好不要用index）

> 使用computed缓存功能

> keep-live缓存组件

```
频繁切换的组件，如tabs
不要乱用，缓存太多会占用内存，且不好debug
```

> 异步组件

```
针对体积较大的组件，如编辑器、复杂表格、复杂表单等
拆包，需要时异步加载，不需要时不加载
减少主包体积，主页加载更快
```

> 路由懒加载

```
项目比较大，拆分路由，保证首页先加载
```

> 服务端渲染SSR

```
可使用Nuxt.js
按需优化，使用SSR的成本比较高
```
